# MiniStock Mobile

Aplicativo de gestão de estoque em React Native + Expo.

## Instalação

```bash
npm install
```

## Como rodar

```powershell
$env:REACT_NATIVE_PACKAGER_HOSTNAME="SEU_IP"
npx expo start --clear
```

Escaneie o QR Code com o Expo Go.

## Tecnologias

- React Native + Expo
- axios
- React Navigation
- AsyncStorage
